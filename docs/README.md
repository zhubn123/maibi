# 文档索引

本目录集中存放麦笔项目的产品、流程、进度和验收文档。根目录仅保留 `README.md` 作为开发入口，以及 `agents.md` 作为 Agent 工作入口。

## 常用文档

- `PLAN.md`：产品范围、核心行为、技术设计、隐私成本控制和测试计划。
- `PR_GUIDELINES.md`：PR 提交规范、免 PR 小改规则、目录约定和推荐拆分顺序。
- `STATUS.md`：当前项目阶段、已完成事项、进行中 PR 和下一步计划。

## 维护规则

- 推进功能 PR 后，必须同步更新 `STATUS.md`。
- 产品边界、核心交互或技术路线变化时，更新 `PLAN.md`。
- 提交流程、PR 拆分方式或多 Agent 协作规则变化时，更新 `PR_GUIDELINES.md` 和根目录 `agents.md`。
